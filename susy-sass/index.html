Susy-SASS is a responsive layout engine for Sass. It is based on Susy 2.x and has been refactored to use SASS instead of node-sass. As Susy is now deprecated, I decided to carry on the project.

Lightweight & Flexible

Copy permalink to “Lightweight & Flexible”

Not everyone can play with the latest specs, and there will always be edge-cases that require manual grid-math. Susy-SASS features: a library of functions that can be used with any other CSS – anywhere, any time.

You can also still access Susy-Sass documentation if necessary.
